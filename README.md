# TweetHarvest

## Description

**TweetHarvest** is a project dedicated to twitter data scraping using keywords. The goal is to collect data from tweets and replies based on keywords.

## Requirements

- Python 3.11
- Python Libraries:
  - Selenium
  - Pandas
